# Facial Recognition: Siamese Network

## Overview

This repository contains the code for facial recognition using a [Siamese network](https://en.wikipedia.org/wiki/Siamese_neural_network). The Siamese network is a type of neural network architecture particularly useful for one-shot learning tasks like facial recognition.

## Features
- **Siamese Network**: Utilizes a Siamese network for facial recognition.
- **One-Shot Learning**: Effective for facial recognition with limited training data.
- **Python Implementation**: Written in Python for ease of use and customization.

## Getting Started
Follow these steps to get the project up and running on your local machine.

### Prerequisites
- Python 3.x

### Installation
1. Clone the repository:

```bash
git clone https://github.com/euphoria-ai/Facial-Recognition.git
```
2. Navigate to the project directory:

```bash
cd Facial-Recognition
```

## Usage
1. [Download the pre-trained model](https://github.com/euphoria-ai/Facial-Recognition) or train your own Siamese network using the provided scripts.
2. Open and run [Facial recognition using Siamese network.ipynb](https://github.com/euphoria-ai/Facial-Recognition/blob/main/Facial%20recognition%20using%20Siamese%20network.ipynb) notebook.
3. This will pop up a camera, input your images and check results.

## Training
If you want to train the Siamese network on your own dataset, follow these steps:

1. Prepare your dataset in the required format.
2. Open the [Facial recognition using Siamese network.ipynb](https://github.com/euphoria-ai/Facial-Recognition/blob/main/Facial%20recognition%20using%20Siamese%20network.ipynb) notebook and follow instructions.
